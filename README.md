# Mandatory chat assignment 1

## Requirements
* [node.js](https://nodejs.org/en/download/)
* [mongodb](https://www.mongodb.com/download-center)

## Installation
* Run `npm install`
* Make sure that you have `mongod` running
* Start the server: `node app.js`
